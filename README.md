# 📈 MarketMind — AI Destekli Kripto & Hisse Analiz

Gerçek zamanlı kripto para ve BIST hisse fiyatlarını çeken, 7 günlük grafik gösteren ve AI ile Türkçe analiz yapan web uygulaması.

## 🚀 Özellikler

- 💰 Gerçek zamanlı kripto fiyatı (CoinGecko API)
- 📊 BIST hisse takibi (Yahoo Finance)
- 📈 7 günlük interaktif fiyat grafiği
- 🤖 AI destekli Türkçe piyasa analizi (Groq API)
- 🌐 Streamlit web arayüzü

## 🛠️ Kullanılan Teknolojiler

- Python 3.11
- Streamlit
- Plotly
- Groq (LLaMA 3.3)
- CoinGecko API
- Yahoo Finance API

## ⚙️ Kurulum

1. Repoyu klonla:
   git clone https://github.com/seydamolla/MarketMind.git
   cd MarketMind

2. Kütüphaneleri kur:
   pip install -r requirements.txt

3. .env dosyası oluştur:
   GROQ_API_KEY=senin_api_keyin

4. Uygulamayı çalıştır:
   streamlit run app.py

## 📌 Desteklenen Semboller

Kripto: bitcoin, ethereum, solana, cardano, dogecoin

BIST Hisse: THYAO.IS, GARAN.IS, POLTK.IS, USAK.IS, ASELS.IS
